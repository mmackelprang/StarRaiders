1. COLOR PALETTE REFERENCE
================================================================================

PRIMARY COLORS
--------------
Black (Space Background)     : #000000  RGB(0, 0, 0)
White (Player/Text)          : #FFFFFF  RGB(255, 255, 255)
Green (HUD Text)             : #00FF00  RGB(0, 255, 0)
Red (Enemies/Danger)         : #FF0000  RGB(255, 0, 0)
Blue (Starbases/Friendly)    : #0000FF  RGB(0, 0, 255)
Cyan (System OK)             : #00FFFF  RGB(0, 255, 255)
Yellow (Warning/Damaged)     : #FFFF00  RGB(255, 255, 0)
Orange (Explosions)          : #FF8800  RGB(255, 136, 0)

SECONDARY COLORS
----------------
Dark Gray (UI Elements)      : #333333  RGB(51, 51, 51)
Light Gray (Disabled)        : #CCCCCC  RGB(204, 204, 204)
Purple (Hyperspace)          : #FF00FF  RGB(255, 0, 255)
Dark Blue (Space Depth)      : #000033  RGB(0, 0, 51)

SYSTEM STATUS COLORS
--------------------
Operational (PESCLR)         : #00FFFF  Cyan
Damaged (PESCLR)             : #FFFF00  Yellow
Destroyed (PESCLR)           : #FF0000  Red

STARFIELD COLORS
----------------
Bright Stars                 : #FFFFFF  White
Medium Stars                 : #CCCCCC  Light Gray
Dim Stars                    : #666666  Dark Gray
Distant Stars                : #333333  Very Dark Gray

2. TYPOGRAPHY SPECIFICATIONS
================================================================================

PRIMARY FONT
------------
Style: Monospace / Fixed-Width (Atari-style)
Recommended: "Courier New", "Consolas", "Monaco", or custom bitmap font
Weight: Regular (400) for body, Bold (700) for emphasis
Size: 12-16pt at 1080p resolution

CHARACTER SET
-------------
Uppercase letters: A-Z
Numbers: 0-9
Symbols: : + - = / [ ] ( ) < > ⊕ ○ ■ ▲ ▼

HUD TEXT SIZING
---------------
Large (Titles):        24pt
Medium (Values):       16pt
Small (Labels):        12pt
Tiny (Indicators):     10pt

ALIGNMENT
---------
HUD Elements:     Left-aligned (left side), Right-aligned (right side)
Center Text:      Titles, Messages, Alerts
Monospace Align:  Ensures consistent column alignment

GRID ELEMENTS:
- Each cell: 16×16 pixel square (adjustable)
- Grid lines: Dark gray (#333333)
- Player cursor (>>): White (#FFFFFF), pulsing
- Starbases (BB): Blue (#0000FF), solid
- Enemies (R/RR): Red (#FF0000), R=single, RR=multiple
- Empty sectors: Black (#000000)

HUD ELEMENTS (Right Side):
- V: Velocity (Green)
- E: Energy (Green, Yellow if <1000, Red if <500)
- K: Kills (Green)
- T: Tracking target (Green, or -- if none)
- TIME: Mission time in centons (Green)
- DC: PESCLR: Damage Control status
  - Letters in Cyan = Operational
  - Letters in Yellow = Damaged
  - Letters in Red = Destroyed

ANIMATED ELEMENTS:
- Player cursor blinks/pulses
- Enemies slowly move between sectors
- Starbase under attack: Flashing red border
- Cursor moves with joystick input

ALERT STATES:
- Starbase surrounded: Flashing red around BB
- Low energy: E value flashes red
- Critical damage: Affected PESCLR letter flashes

    ▲

COLOR CODING:
- Red (#FF0000) for all enemy ships
- Brighter red when closer
- Dimmer/smaller when farther

TORPEDO TRAIL:
═════════► (White line with arrow head)
- Animated movement toward target
- Fades after 0.5 seconds

POSITION INDICATORS:
○ Horizontal Position (Left): Shows left/right alignment
○ Vertical Position (Right): Shows up/down alignment
- White circles (#FFFFFF)
- Filled when aligned

LOCK INDICATORS:
⊕ Three crosshair symbols at bottom
- Green (#00FF00) when locked
- Gray (#666666) when not locked
- All three green = optimal firing

RANGE DISPLAY:
R: 45 - Distance to target in metrons
- Green: 30-70 (optimal range)
- Yellow: 15-29 or 71-90 (acceptable)
- Red: <15 or >90 (poor range)

HUD STATUS (Top):
V: Velocity (0-9, speed level)
E: Energy (current value, color-coded)
K: Kills (total destroyed)
T: Tracking target (F=Fighter, C=Cruiser, B=Basestar)

BOTTOM STATUS BAR:
[S] SHIELDS: ON/OFF - Green when ON, Gray when OFF
[T] COMPUTER: ON/OFF - Green when ON, Gray when OFF
Navigation shortcuts displayed

VISUAL EFFECTS:
- Shield shimmer: Subtle blue glow around screen edges when active
- Hit flash: Screen flash red when taking damage
- Explosion: Expanding orange/red circle with debris
- Screen shake: Brief shake on collision

12. ANIMATION SEQUENCES
================================================================================

TORPEDO FIRE SEQUENCE:
Frame 1: Muzzle flash at ship position (bright white circle)
Frame 2: Torpedo appears (white line, 10 pixels long)
Frame 3-10: Torpedo extends (═════════►), moves toward target
Frame 11: Hit detection
Frame 12: If hit - explosion; if miss - torpedo fades out
Duration: ~0.5 seconds total

EXPLOSION SEQUENCE:
Frame 1: Bright white flash at impact point
Frame 2: Orange circle expands (radius 10px)
Frame 3: Orange circle expands (radius 20px), turns red
Frame 4: Red circle expands (radius 30px), adds debris (small dots)
Frame 5: Debris spreads outward, circle fades
Frame 6-8: Debris continues spreading, fading to black
Duration: ~1.0 second total

HYPERSPACE ENTRY:
Frame 1-5: Stars begin stretching into lines
Frame 6-10: Lines accelerate outward
Frame 11-20: Full warp tunnel effect
Frame 21-25: Deceleration, lines shorten
Frame 26-30: Return to normal starfield
Duration: ~3-5 seconds total

SHIELD ACTIVATION:
Frame 1: Blue glow appears at screen edges
Frame 2-3: Glow pulses brighter
Frame 4+: Steady shimmer effect (subtle pulse)
Continuous: While shields active

DAMAGE FLASH:
Frame 1: Screen flash red (50% opacity)
Frame 2: Screen shake (2 pixels random direction)
Frame 3: Red flash fades
Frame 4: Return to normal
Duration: ~0.2 seconds

STARBASE UNDER ATTACK ALERT:
Frame 1-10: Starbase icon flashes red (on)
Frame 11-20: Starbase icon normal (off)
Repeat: Until threat cleared or starbase destroyed
Frequency: 1 blink per second

ENEMY MOVEMENT (Galactic Chart):
Frame 1: Enemy in sector (X,Y)
Frame 2-30: Slow fade/transition animation
Frame 31: Enemy in adjacent sector
Duration: 2-3 seconds per move
Frequency: Every 10-20 centons (game time)

LOCK INDICATOR ANIMATION:
Unlocked: Gray ⊕ symbol (static)
Locking: Yellow ⊕ symbol (pulsing)
Locked: Green ⊕ symbol (solid, slight pulse)
Transition: 0.3 seconds

13. VISUAL EFFECTS GUIDE
================================================================================

STARFIELD PARALLAX:
- Layer 1 (Near): Large white dots, scroll speed 100%
- Layer 2 (Mid): Medium gray dots, scroll speed 50%
- Layer 3 (Far): Small dark gray dots, scroll speed 25%
- Layer 4 (Distant): Tiny dim dots, scroll speed 10%
- Direction: Based on ship velocity and heading
- Density: 100-200 stars visible at any time

SHIELD SHIMMER EFFECT:
- Translucent blue overlay at screen edges
- Opacity: 20-30% (semi-transparent)
- Animation: Subtle wave pattern moving inward
- Frequency: 2-3 waves per second
- Intensity increases when taking hits

SCREEN SHAKE (Damage):
- Displacement: ±2-4 pixels random direction
- Duration: 1-3 frames (0.017-0.05 seconds)
- Trigger: Any damage received
- Severity scales with damage amount

TORPEDO TRAIL:
- Base: White line (═══════)
- Head: Bright arrow (►)
- Tail: Fading gradient (white → gray → transparent)
- Length: 40-60 pixels
- Speed: 50 metrons/second visual
- Glow: Slight bloom effect around trail

EXPLOSION DEBRIS:
- Particles: 8-12 small triangles/squares
- Color: Orange (#FF8800) → Red (#FF0000) → Dark Gray (#333333)
- Velocity: Radial outward from explosion center
- Lifespan: 0.5-1.0 seconds
- Rotation: Random spin on each particle

HYPERSPACE TUNNEL:
- Star streaks: Vertical lines stretching from center
- Colors: White → Cyan → Purple → Magenta (cycling)
- Speed: Accelerates during entry, decelerates at exit
- Density: Increases toward edges (tunnel effect)
- Glow: Bloom effect on bright streaks

ENEMY APPROACH/RETREAT:
- Scale: Enemy size increases as distance decreases
- Max size: 60×60 pixels (very close)
- Min size: 4×4 pixels (distant)
- Smoothing: Linear interpolation between positions
- Rotation: Slight banking when turning

LOW ENERGY WARNING:
- Effect: Energy value flashes red
- Frequency: 2 flashes per second when E < 500
- Increase: 4 flashes per second when E < 100
- Audio: Accompanying beep tone

CRITICAL SYSTEM DAMAGE:
- Effect: Damaged PESCLR letter flashes
- Color: Yellow ↔ Red (alternating)
- Frequency: 1 flash per second
- Indicator: Visual alert in HUD

GALACTIC CHART CURSOR:
- Symbol: >> (double chevron)
- Color: White (#FFFFFF)
- Animation: Pulse brightness (75% → 100% → 75%)
- Frequency: 1 pulse per second
- Movement: Smooth interpolation between sectors

DOCKING SEQUENCE:
- Frame 1-5: Starbase grows larger (approaching)
- Frame 6: White flash (docking confirmation)
- Frame 7-10: Fade to black
- Frame 11: "DOCKED AT STARBASE" message
- Frame 12: Transition to Galactic Chart
- Duration: ~2 seconds total

SYSTEM REPAIR ANIMATION (at Starbase):
- Each PESCLR letter sequentially changes:
  Red → Yellow → Cyan (0.2 second per letter)
- Accompanied by "repair" sound effect
- Energy bar fills simultaneously
- Total duration: ~1.5 seconds

- Significant letterboxing
- Maintain 16:9 aspect for gameplay
- ~12% black bars (sides or top/bottom)

MOBILE (PORTRAIT):
- Not recommended for original design
- If implemented: Rotate prompts or simplified UI
- Consider touch controls overlay

MOBILE (LANDSCAPE):
- Acceptable but cramped on small screens
- Minimum 7" tablet recommended
- Touch controls replace keyboard
- Simplified HUD (smaller text, fewer details)

SCALING GUIDELINES:
- Below 720p: Use pixel-perfect scaling (2×)
- 720p-1080p: Use smooth scaling
- Above 1080p: Use high-DPI assets
- Always maintain 16:9 gameplay aspect ratio
